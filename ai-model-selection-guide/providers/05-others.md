## Perplexity Pro

**Best For**: Research, fact-finding, current events
**Price**: $20/month
**Key Feature**: Citations for every answer

### What You Get

- ✅ **Always-on web search** (every answer includes sources)
- ✅ **600 Pro searches/day** (uses GPT-4, Claude, or their own model)
- ✅ **Unlimited Quick searches** (faster, basic model)
- ✅ **File upload** (PDF, docs for analysis)
- ✅ **Image generation** (DALL-E 3, Flux)
- ✅ **Focus modes** (Academic, Writing, Math, etc.)
- ❌ **No long document editing** (not a writing tool)

### When to Choose Perplexity

**✅ Choose Perplexity When**:
- Research is your primary use case (80%+ of usage)
- You need citations for every claim
- You work with current events, news, trends
- You value accuracy over creative writing

**❌ Choose ChatGPT/Claude Instead When**:
- You need all-around tool (writing, code, images)
- Citations aren't critical
- You need long document editing

### Use Case Example

**Task**: Research current AI regulation in Southeast Asia

**Perplexity**:
```
Answer: Singapore updated its AI Governance Framework in Jan 2026...
Sources:
[1] Singapore IMDA - AI Governance Framework 2026
[2] Malaysia MDEC - National AI Roadmap Update
[3] Reuters - SEA AI regulation overview
```

**ChatGPT** (with web search):
```
Answer: Singapore has updated its AI governance framework...
(Shows sources, but less prominent)
```

**Winner**: Perplexity for dedicated research

## Microsoft Copilot

**Best For**: Microsoft 365 users (Word, Excel, PowerPoint, Teams)
**Price**: $20/month (Pro) or $30/user/month (for Microsoft 365)

### Available Plans

| Plan | Price | Features |
|------|-------|----------|
| **Copilot Free** | $0 | Limited, Bing Chat with GPT-4 |
| **Copilot Pro** | $20/mo | Priority access, image generation, some Office integration |
| **Copilot for Microsoft 365** | $30/user/mo* | Full Office integration (Word, Excel, PowerPoint, Teams, Outlook) |

*Requires Microsoft 365 subscription (additional $12-22/user/month)

### Copilot for Microsoft 365

**What It Does**:
- **Word**: Draft documents, summarize, rewrite
- **Excel**: Analyze data, create formulas, generate charts
- **PowerPoint**: Create presentations from prompts
- **Teams**: Summarize meetings, generate action items
- **Outlook**: Draft emails, summarize threads

**Total Monthly Cost**:
- Microsoft 365 Business: $12.50/user/month
- Copilot for 365: $30/user/month
- **Total**: $42.50/user/month

### When to Choose Microsoft Copilot

**✅ Choose Copilot When**:
- You're already a Microsoft 365 customer
- You live in Word, Excel, PowerPoint daily
- You need enterprise compliance (Microsoft ecosystem)
- Your team is 10+ people (better ROI at scale)

**❌ Choose ChatGPT/Claude Instead When**:
- You don't use Microsoft 365
- Budget-conscious ($20 vs $42.50/user)
- You use Google Workspace
- Need better writing quality (Claude) or features (ChatGPT)

## Mistral AI

**Best For**: European data residency, multilingual, cost-effective API
**Price**: API only (no consumer subscription)

### Available Models

| Model | Input $/1M | Output $/1M | Context | Best For |
|-------|------------|-------------|---------|----------|
| **Mistral Large** | $2.00 | $6.00 | 128K | General purpose, multilingual |
| **Mistral Small** | $0.20 | $0.60 | 32K | Simple tasks, high volume |
| **Mistral Nemo** | $0.15 | $0.15 | 128K | Budget option |

### Key Differentiators

1. **European AI** - Data processed in EU (GDPR compliant)
2. **Multilingual** - Excellent at French, Spanish, German, Italian
3. **Cost-effective** - Cheaper than OpenAI/Anthropic for similar quality
4. **Open source** - Some models (Mistral 7B) are open source

### When to Choose Mistral

**✅ Choose Mistral When**:
- You need EU data residency
- Multilingual content (European languages)
- Building applications with API (cost-sensitive)
- Want open-source option

**❌ Choose OpenAI/Anthropic Instead When**:
- Need consumer interface (Mistral is API-only for most use cases)
- English-only content
- Need web search, images, or other features

## Jasper AI

**Best For**: Marketing content, SEO optimization, brand voice
**Price**: $49/month (Creator), $125/month (Teams)

### What Jasper Adds

**vs ChatGPT Plus ($20/month)**:
- ✅ **SEO optimization** - Built-in keyword research, content scoring
- ✅ **Brand voice** - Train on your writing style
- ✅ **Templates** - 50+ marketing templates (ads, emails, blogs)
- ✅ **Grammarly integration** - Grammar/style checking
- ❌ **2.5x more expensive**

### When to Choose Jasper

**✅ Choose Jasper When**:
- You're a marketing agency or content team
- SEO is critical (need keyword optimization)
- You produce 50+ pieces of content/month
- Budget allows $49-125/month

**❌ Choose ChatGPT Instead When**:
- Budget is $20/month
- You can do SEO optimization manually
- Solo content creator (Jasper is overkill)

**Cost-Benefit**:
- Jasper: $49/month (SEO features, brand voice)
- ChatGPT Plus: $20/month + manual SEO = save $29/month
- Break-even: If Jasper saves >2 hours/month on SEO, worth it

## Otter.ai

**Best For**: Meeting transcription, Zoom/Google Meet integration
**Price**: $10/month (Pro), $20/month (Business)

### What Otter Does

- ✅ **Automatic transcription** - Zoom, Google Meet, Teams
- ✅ **Speaker identification** - Labels who said what
- ✅ **Action items** - Automatically extracts tasks
- ✅ **Search** - Find moments in recordings
- ✅ **Highlights** - Key points and decisions

### Alternatives

| Tool | Price | Best For |
|------|-------|----------|
| **Otter.ai** | $10/mo | Dedicated transcription, best accuracy |
| **ChatGPT Plus** | $20/mo | Upload audio → transcribe (less accurate) |
| **Fireflies.ai** | $10/mo | Similar to Otter, alternative |
| **Fathom** | Free | Free alternative (good accuracy) |

### When to Choose Otter

**✅ Choose Otter When**:
- You have 5+ meetings/week
- Need automatic transcription (don't want to upload manually)
- Value accuracy (Otter is best-in-class)
- Budget allows $10/month

**❌ Choose Free Alternative When**:
- <5 meetings/week
- Can manually upload to ChatGPT
- Budget is tight

**Best For**: High-quality artistic images, creative visuals
**Price**: $10/month (Basic), $30/month (Standard), $60/month (Pro)

### Midjourney vs DALL-E 3 (in ChatGPT)

| Feature | Midjourney | DALL-E 3 (ChatGPT Plus) |
|---------|-----------|------------------------|
| **Price** | $10/mo | $20/mo (includes ChatGPT) |
| **Quality** | ⭐⭐⭐⭐⭐ Artistic, stylized | ⭐⭐⭐⭐ Realistic, literal |
| **Ease** | Discord bot (learning curve) | Text interface (easy) |
| **Speed** | 1-2 min/image | 30 sec/image |
| **Commercial use** | ✅ Yes (on paid plans) | ✅ Yes |
| **Best for** | Art, creative visuals, marketing | Quick mockups, illustrations |

### When to Choose Midjourney

**✅ Choose Midjourney When**:
- Creating marketing visuals, brand assets
- Need artistic, stylized images
- Willing to learn Discord interface
- Budget allows $10/month

**❌ Choose DALL-E 3 (ChatGPT) When**:
- Need quick, simple images
- Want text-to-image in same interface
- Don't want to use Discord
- Need all-around tool (Midjourney is images only)

## Open Source Models

**Best For**: Self-hosting, privacy, no API costs, customization
**Price**: Free (models) + hosting costs

### Popular Open Source Models

| Model | Size | Provider | Best For |
|-------|------|----------|----------|
| **Llama 3.1** | 8B, 70B, 405B | Meta | General purpose |
| **Qwen 2.5** | 7B, 32B, 72B | Alibaba | Multilingual, code |
| **Mistral 7B** | 7B | Mistral | European languages |
| **DeepSeek V3** | 671B | DeepSeek | Code generation |

### Self-Hosting Options

#### Option 1: Ollama (Local Mac/PC)
**Cost**: $0 (runs on your computer)
**Pros**: Maximum privacy, no API costs
**Cons**: Limited to smaller models (8B-32B), slower

```bash
# Install Ollama
curl -fsSL https://ollama.ai/install.sh | sh

# Run Llama 3.1
ollama run llama3.1

# Use in terminal
> "Write me an email about..."
```

#### Option 2: VPS Hosting (Oracle, Contabo)
**Cost**: $0-10/month
**Pros**: More powerful, 24/7 access
**Cons**: Requires technical setup

**Oracle Free Tier**:
- Free forever (4 ARM cores, 24GB RAM)
- Can run 13B-32B models
- See [OpenClaw SEA Guide](../../openclaw-sea-guide/) for setup

#### Option 3: API (Groq, Together.ai)
**Cost**: $0-5/month for moderate use
**Pros**: Fast inference, no hosting
**Cons**: Not truly self-hosted

### When to Choose Open Source

**✅ Choose Open Source When**:
- Maximum privacy required (healthcare, legal)
- Building custom applications
- High volume (>1M tokens/month makes API expensive)
- Technical team available

**❌ Choose Commercial (OpenAI/Anthropic) When**:
- Need best quality (405B Llama < GPT-4)
- Non-technical user
- Low volume (<100K tokens/month)
- Need features (web search, images)
