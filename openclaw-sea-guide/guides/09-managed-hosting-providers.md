## Security Best Practices (ALL Providers)

Regardless of which provider you choose, **always** follow these rules:

### 1. Use Isolated API Keys
- ‚ùå **NEVER** use your primary OpenAI or Claude account
- ‚úÖ **CREATE** a new, separate API key for OpenClaw only
- ‚úÖ **SET** spending limits ($20/month maximum)

### 2. Use Throwaway Credentials
- ‚ùå **NEVER** give OpenClaw your main email password
- ‚ùå **NEVER** give OpenClaw admin access to business systems
- ‚úÖ **CREATE** separate, limited-access accounts

### 3. Enable All Security Features
- ‚úÖ **ENABLE** DM pairing (prevents strangers from using your bot)
- ‚úÖ **ENABLE** authentication
- ‚úÖ **DISABLE** public access
- ‚úÖ **USE** HTTPS only

### 4. Monitor Usage
- ‚úÖ **CHECK** API usage weekly
- ‚úÖ **REVIEW** OpenClaw logs monthly
- ‚úÖ **WATCH** for unusual activity

### 5. Keep Updated
- ‚úÖ **UPDATE** OpenClaw immediately when security patches are released
- ‚úÖ **SUBSCRIBE** to OpenClaw security announcements
- ‚úÖ **MONITOR** CVE databases
